Post 2.6

Preparacion final

Preparacion

Este postre llamado Cheesecake consta de tres capas, la primera la masa la cual es la base de esta preparación y esta formada a grandes rasgos por galleta triturada. La segunda capa, la crema esta capa es una mezcla de yogurt natural, leche condensada y otros ingredientes que ayudan a su aglutinación, por ultimo la punta esta echa de una mezcla de frambuesas y gelatina de sabor. Esta receta es una variación de la receta original del pastel de queso, la cual es mucho mas elaborada y compleja además de necesitar el uso de un horno, cosa que se cambio en esta receta para lograr hacerla con el molde fabricado.

Trabajo Fotogrametria

En este trabajo de fotogrametría se puede ver la cuarta interacción con el programa de Recap, esta interacción no alcanza los resultados deseados ya que las texturas no quedan reflejadas en el modelo digital. Además si bien se aprecian los colores de la preparación las texturas no reflejan la real apariencia del platillo. Esta interacción pudiera ser considerada como un fracaso, pero mediante esta pude entender mejor como funciona el programa y como solucionar los errores cometidos para las entregas que vengan mas adelante.

Trabajo de molde

En esta fase quise llevar la preparación a un terreno más concreto, de esta manera mediante el programa de Fusion 360, diseñe un molde el cual luego podria ser impreso. Este modelo con forma de piramide hexagonal permitiría poder hacer esta preparación múltiples veces. Además cabe destacar que este diseño presenta un angulo de desmolde muy pequeño por lo que al momento del desmolde surgen ciertas complicaiones.

Molde real

Para este molde se uso la impresion 3D, de esta manera se pudo llevar el diseño digital a algo tangible a lo que se le pudo dar utilidad en la cocina. Este molde si bien cumple con su funcion tiene algunas complicaciones a la hora de su uso, al no poder ser sometido a altas temperaturas.
