Preparacion final

Trabajo de fotografia

Preparacion

Este postre llamado Cheesecake consta de tres capas, la primera la masa la cual es la base de esta preparación y esta formada a grandes rasgos por galleta triturada. La segunda capa, la crema esta capa es una mezcla de yogurt natural, leche condensada y otros ingredientes que ayudan a su aglutinación, por ultimo la punta esta echa de una mezcla de frambuesas y gelatina de sabor. Esta receta es una variación de la receta original del pastel de queso, la cual es mucho mas elaborada y compleja además de necesitar el uso de un horno, cosa que se cambio en esta receta para lograr hacerla con el molde fabricado.

Trabajo Fotogrametria

En este trabajo de fotogrametría se puede ver la tercera interacción con el programa de Recap, esta interacción no alcanza los resultados deseados ya que las texturas no quedan reflejadas en el modelo digital. Además si bien se aprecian los colores de la preparación las texturas no reflejan la real apariencia del platillo. Esta interacción pudiera ser considerada como un fracaso, pero mediante esta pude entender mejor como funciona el programa y como solucionar los errores cometidos para las entregas que vengan mas adelante.

Trabajo de molde

En esta fase quise llevar la preparación a un terreno más concreto, de esta manera mediante el programa de Fusion 360, diseñe un alcance a lo que podría ser un modelo funcional. Este modelo con forma piramidal permitiría poder hacer esta preparación múltiples veces ya que estaría echo de acero, material resistente y duradero, además cabe destacar que el Angulo de desmolde permitiría poder sacar la preparación con facilidad.

Molde real

Para este molde use mica y mediante folding pude darle la forma deseada. Si bien este material cumple su función la cual es dar la forma a la preparación, al estar echo de este material limita que las preparaciones solo puedan ser echas en frio, ya que por razones obvias seria imposible someterlo a altas temperaturas.
